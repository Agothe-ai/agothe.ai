# Couture Transformation Phase Images

Drop transformation phase images (.webp format) into this directory. These represent the 4 transformation phases with δ_H values.

## Required Files:
- `transformation-phase-1-substrate.webp` → Phase 1
- `transformation-phase-2-activation.webp` → Phase 2
- `transformation-phase-3-emergence.webp` → Phase 3
- `transformation-phase-4-integration.webp` → Phase 4

## Guidelines:
- Format: WebP
- Visual representations of transformation stages
- Should convey evolution/progression
- Referenced by `lib/couture-data.ts`
